# TutorPro - Professional Tuition Service Website

TutorPro is a modern, responsive tuition service website that provides comprehensive information about tutoring services, courses, and contact options.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Multiple Pages**: Home, About, Services, and Contact pages
- **Interactive Navigation**: Mobile-friendly hamburger menu
- **Contact Form**: Functional contact form with validation
- **Modern UI**: Clean and professional design with smooth animations
- **Course Catalog**: Comprehensive listing of subjects and tutoring formats

## Pages

### Home (index.html)
- Hero section with call-to-action
- Features highlighting key benefits
- Popular subjects showcase
- Call-to-action section

### About (about.html)
- Mission and story
- Core values
- Team information
- Statistics and achievements

### Services (services.html)
- Academic subjects listing
- Test preparation programs
- Learning format options
- Pricing information

### Contact (contact.html)
- Contact form with validation
- Contact information
- Business hours
- Social media links

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript**: Interactive features and form validation
- **Responsive Design**: Mobile-first approach

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/milanjyotiray/TutorPro.git
```

2. Open the project folder:
```bash
cd TutorPro
```

3. Open `index.html` in your web browser:
```bash
# On Linux/Mac
open index.html

# On Windows
start index.html
```

Or simply double-click the `index.html` file to open it in your default browser.

## Customization

### Colors
The main color scheme can be customized in `css/style.css`:
- Primary color: `#3498db` (blue)
- Secondary color: `#667eea` (purple)
- Dark color: `#2c3e50` (dark blue)

### Content
- Update text content directly in the HTML files
- Modify styles in `css/style.css`
- Add new pages following the existing structure
